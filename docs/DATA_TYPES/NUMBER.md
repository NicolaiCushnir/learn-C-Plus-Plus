Что значит `signed` и `unsigned` ?
* `signed` — знак есть, то есть число может быть отрицательным или положительным.
* `unsigned` — знака нет, число может быть только положительным (или ноль).

### Exemples :

* Type of data `integer`.
```C++
int age = 28;
```

* Type of data `float`.
```C++
float area = 64.74;
```

* Type of data `double`.
```C++
double c = 19.3;
double volume = 134.64534;
```

* Type of data `short int`.
```C++
#include <iostream>

int main () {
	std::cout << "Type of data - 'short int' \n";

	short int salary = 3999;
	std::cout << salary;
}
```
