### [return main page](../README.md) || Topic. Check if you instaled C++ :

If you want to check if C++ is instaled on your computer then use in console or terminal :
```bash
g++ --version
```

Также можно проверить GCC (общий компилятор)
```bash
gcc --version
``` 

Можно ищё сделать где находится компилятор :
```bash
where g++

# Есть ищё такой вормат, по идее путь без exe file
# А так похожие команды. (where && which)
which g++
```
